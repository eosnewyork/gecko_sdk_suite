46 // The basic TOKEN_DEF macro should not be used directly since the simplified
47 // definitions are safer to use. For completeness of information, the basic
48 // macro has the following format:
49 //
50 // TOKEN_DEF(name,creator,iscnt,isidx,type,arraysize,...)
51 // name - The root name used for the token
52 // creator - a "creator code" used to uniquely identify the token
53 // iscnt - a boolean flag that is set to identify a counter token
54 // isidx - a boolean flag that is set to identify an indexed token
55 // type - the basic type or typdef of the token
56 // arraysize - the number of elements making up an indexed token
57 // ... - initializers used when reseting the tokens to default values
58 //
59 //
60 // The following convenience macros are used to simplify the definition
61 // process for commonly specified parameters to the basic TOKEN_DEF macro
62 // DEFINE_BASIC_TOKEN(name, type, ...)

64 // DEFINE_COUNTER_TOKEN(name, type, ...)
